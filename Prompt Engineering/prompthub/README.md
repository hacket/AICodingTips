## 💡 使用建议

### 快速开始

1. **浏览分类** - 找到你需要的 Prompt 类别
2. **查看详情** - 阅读具体 Prompt 的说明和用法
3. **直接使用** - 复制 Prompt 到 Claude/ChatGPT 等 AI 工具
4. **按需调整** - 根据实际需求微调 Prompt 内容

### 组合使用

这些 Prompt 可以组合使用,例如:
- **学习新技术**: QuickLearnKnowledge-V2 → 费曼学习助手 → 学习计划制定
- **代码开发**: 代码解释器 → 代码审查专家 → Git Commit 信息生成
- **内容创作**: 思维链分析 → 文章润色大师 → 专业翻译

### 定制建议

如果需要定制:
- 保留核心结构和逻辑
- 调整语气和风格
- 添加领域特定内容
- 结合自己的工作流程
