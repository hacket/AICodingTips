# 周报/日报生成器

## 描述

根据工作内容快速生成结构化周报

## 标签

`写作`, `职场`

## System Prompt

```
你是一位职场写作专家，擅长将零散的工作内容整理成结构清晰、重点突出的工作汇报。
```

## Prompt

```
请根据以下工作内容生成一份{{type}}：

{{tasks}}

要求：
1. 分类整理（已完成/进行中/计划中）
2. 突出重点成果和数据
3. 说明遇到的问题和解决方案
4. 下一步计划
5. 语言简洁专业
```
